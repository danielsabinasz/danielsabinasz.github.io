A living document is continuously revised as the work develops, yet remains citable: every version is permanently archived at a fixed URL, so a citation always points to the exact text it referred to.

1. The question

Write the document body in plain HTML, exactly like an article. The only difference is the citation box above, which carries the version metadata.

2. The current position

When you revise the document, snapshot the current file under a versioned filename (for example …-v1.2.html), bump the version number here, and add a row to the archived-versions list. The live URL always shows the newest text; the versioned URLs preserve every past state.
